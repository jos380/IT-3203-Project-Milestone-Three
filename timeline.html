Web Development Timeline

1989

World Wide Web was created by Tim Berners-Lee at CERN. He developed the first web browser, server, and the initial version of HTML.

1995

HTML 2.0 was standardized, introducing significant improvements including support for forms and table structures thanks to respective tags written in the source code.

1998

CSS2 was developed by the W3C, adding support for media-specific style sheets, downloadable fonts, element positioning, and tables.

2004

The term "Web 2.0" gained popularity, representing a shift toward user-generated content, usability, and interoperability for end users.
